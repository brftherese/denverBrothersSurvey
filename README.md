# denverBrothersSurvey

## GitHub Pages Setup

This repository is configured to be hosted on GitHub Pages.

### How to Enable GitHub Pages

1. Go to your repository settings: `https://github.com/brftherese/denverBrothersSurvey/settings/pages`
2. Under "Source", select the branch you want to deploy (typically `main` or `master`)
3. Select the folder (use `/ (root)` for files in the root directory)
4. Click "Save"

Once enabled, your site will be available at: `https://brftherese.github.io/denverBrothersSurvey/`

### Customizing Your Webpage

Replace the content in `index.html` with your actual webpage content. You can also add additional HTML, CSS, and JavaScript files as needed.
